What is actually HelbManager2022 ?

HelbManager2022 is a web application in Django that allows users to manage projects through a simplified task scheduling system.




This web application offers several features such as :

  • A Registration/Login System on the platform.

  • A Profile System for Registered users.

  • A System for Creating and Managing Projects as described in this description given in this statement (project, task, subtasks, status).

  • A System for Registering users to a Project created by a Project Manager.

  • A "Drag & Drop" Interface to notify the Status of Tasks and Subtasks of the Project.

  • A Notifications System when the Status of a task or sub-task changes.

  • An Interface that allows to graphically visualize the chronology of Projects progress.
